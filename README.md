# 管用料計算工具

這是一個用於計算管用料的網頁應用程式。

## 功能

*   從 Excel (.xlsx) 檔案匯入料單。
*   自動計算所需料件。
*   將計算結果分組和處理。
*   將最終結果匯出為 Excel 檔案。

## 如何使用

1.  複製或下載此儲存庫中的所有檔案。
2.  在您的網頁瀏覽器中打開 `管用料計算工具.html` 檔案。
3.  依照頁面指示上傳您的 Excel 料單。
4.  點擊計算並下載結果。
